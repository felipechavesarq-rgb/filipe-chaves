# Gerenciamento de Estoque

Este é um sistema de **controle de estoque** para uma organização fictícia. O sistema permite o cadastro de fornecedores, produtos e a associação entre eles, garantindo eficiência e rastreabilidade nas operações de estoque.

## Funcionalidades
- **Cadastro de Fornecedores:** Gerencia as informações dos fornecedores, como nome, CNPJ, telefone e e-mail.
- **Cadastro de Produtos:** Gerencia os produtos em estoque, incluindo nome, código de barras, descrição e quantidade.
- **Associação de Fornecedores aos Produtos:** Permite associar um fornecedor a um produto para rastrear as compras e relações de fornecimento.

## Tecnologias Usadas:
- Backend: **Node.js** com **Express.js**
- Banco de Dados: **MongoDB**
- Frontend: **React.js**
